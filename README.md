# Hello Android - Jetpack Compose

Repositori ini berisi implementasi aplikasi Android sederhana "Hello Android" menggunakan **Jetpack Compose**. Proyek ini dikembangkan sebagai bagian dari tugas mata kuliah Pemrograman Perangkat Bergerak (PPB).

## 📱 Deskripsi Proyek
Aplikasi ini adalah pengenalan dasar terhadap framework deklaratif UI Android, Jetpack Compose. Aplikasi akan menampilkan teks "Hello Android!" yang diposisikan tepat di tengah layar menggunakan layout `Box` dan modifier `Alignment.Center`.

## 🛠️ Teknologi yang Digunakan
* **Bahasa Pemrograman:** Kotlin
* **UI Framework:** Jetpack Compose (Material Design 3)
* **IDE:** Android Studio
* **Minimum SDK:** API 24 (Android 7.0 Nougat)

## 🚀 Cara Menjalankan Aplikasi

Aplikasi ini dikembangkan menggunakan **Android Studio**. Untuk menjalankan proyek ini dan melihat hasil dari `MainActivity.kt`, ikuti panduan berikut:

### 1. Buka Proyek di Android Studio
1. Buka aplikasi **Android Studio**.
2. Klik **Open** (atau tap **File > Open** dari menu bar).
3. Cari dan pilih direktori/folder proyek `Tugas-PPB-HelloAndroid` atau repositori yang telah di-clone, lalu tekan **OK**.
4. Tunggu beberapa saat agar proses **Gradle Sync** selesai (perhatikan bar loading di bagian bawah layar).

### 2. Memeriksa MainActivity.kt
File inti pada aplikasi ini berada di direktori berikut:
`app/src/main/java/com/fazdy/helloandroid/MainActivity.kt`

Anda dapat membukanya melalui panel **Project** di sebelah kiri layar Android Studio untuk memeriksa atau memodifikasi kode.

### 3. Build & Run
Untuk menjalankan aplikasi di emulator atau perangkat fisik:
1. Pastikan Anda sudah menyiapkan **Android Virtual Device (AVD)** atau menyambungkan perangkat Android fisik yang sudah mengaktifkan *USB Debugging*.
2. Pilih perangkat/emulator tujuan pada dropdown di sebelah tombol Run pada toolbar atas Android Studio.
3. Klik tombol **Run 'app'** (ikon segitiga hijau ▶️) atau gunakan shortcut `Control + R` (Mac) / `Shift + F10` (Windows).
4. Gradle akan mulai mem-build aplikasi (`Building...`). Jika berhasil, aplikasi akan otomatis terbuka di perangkat genggam atau emulator Anda, memperlihatkan hasil dari eksekusi `MainActivity`.
